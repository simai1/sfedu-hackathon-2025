# Инструкция по распространению приложения

## Подготовка дистрибутива

### Для Windows

1. **Соберите приложение:**
   ```cmd
   build.bat
   ```

2. **Создайте дистрибутив:**
   ```cmd
   create_distribution.bat
   ```

3. **Результат:**
   - Папка `distribution/BitApp/` с файлами
   - Архив `distribution/BitApp.zip` (если установлен 7-Zip)

### Для macOS

1. **Соберите приложение:**
   ```bash
   ./build_macos.sh
   ```

2. **Создайте .app пакет (опционально):**
   ```bash
   ./create_macos_app.sh
   ```

3. **Создайте дистрибутив:**
   ```bash
   ./create_distribution.sh
   ```

## Что входит в дистрибутив

- `BitApp.exe` (Windows) или `BitApp` (macOS/Linux) - исполняемый файл
- `README.txt` - подробная инструкция для пользователей
- `ИНСТРУКЦИЯ.txt` - краткая инструкция

## Важные моменты

### Безопасность

1. **Подпись кода (рекомендуется):**
   - Для Windows: используйте код-подпись с сертификатом
   - Для macOS: используйте Apple Developer ID
   - Это уменьшит предупреждения безопасности

2. **Антивирусы:**
   - Неподписанные приложения могут блокироваться
   - Рекомендуется добавить приложение в исключения популярных антивирусов

### Обновления

1. **Версионирование:**
   - Используйте версии в названии файла: `BitApp-v1.0.zip`
   - Обновляйте версию при каждом релизе

2. **Уведомления:**
   - Рассмотрите добавление проверки обновлений в приложение
   - Или создайте RSS/JSON feed с информацией об обновлениях

### Поддержка пользователей

1. **Документация:**
   - Подробная инструкция в README.txt
   - FAQ на веб-сайте
   - Видео-инструкции (опционально)

2. **Обратная связь:**
   - Email для поддержки
   - Форум или чат
   - Система тикетов

## Альтернативные варианты распространения

### Установщик (Windows)

Можно создать установщик с помощью:
- **Inno Setup** (бесплатный)
- **NSIS** (бесплатный)
- **WiX Toolset** (бесплатный, от Microsoft)

### App Store (macOS)

Для распространения через Mac App Store:
- Требуется Apple Developer аккаунт ($99/год)
- Нужна подпись кода
- Требуется соответствие правилам App Store

### Microsoft Store (Windows)

Для распространения через Microsoft Store:
- Требуется аккаунт разработчика ($19/раз или $99/год)
- Нужна подпись кода
- Требуется упаковка в формат MSIX
