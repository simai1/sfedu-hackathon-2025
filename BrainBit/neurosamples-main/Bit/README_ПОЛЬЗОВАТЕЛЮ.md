# Bit App - Инструкция для пользователей

## Системные требования

- **ОС:** Windows 10/11 (64-bit)
- **Дополнительно:** Visual C++ Redistributable (обычно уже установлен)

## Установка и запуск

### Шаг 1: Скачайте приложение

Скачайте архив `BitApp.zip` с веб-сайта и распакуйте его в любую папку на вашем компьютере.

### Шаг 2: Запуск приложения

1. Откройте папку, куда вы распаковали архив
2. Найдите файл `BitApp.exe`
3. Дважды кликните на `BitApp.exe` для запуска

**Важно:** При первом запуске Windows может показать предупреждение о неизвестном издателе. Это нормально для неподписанных приложений.

### Если Windows блокирует запуск

Если Windows блокирует запуск приложения:

1. **Windows SmartScreen:**
   - Нажмите "Подробнее"
   - Нажмите "Выполнить в любом случае"

2. **Антивирус:**
   - Некоторые антивирусы могут блокировать неподписанные приложения
   - Добавьте папку с приложением в исключения антивируса

3. **Блокировка файла:**
   - Кликните правой кнопкой на `BitApp.exe`
   - Выберите "Свойства"
   - Внизу окна нажмите "Разблокировать" (если доступно)
   - Нажмите "ОК"

## Использование приложения

1. При запуске откроется окно поиска устройств
2. Включите ваше устройство BrainBit
3. Дождитесь, пока устройство появится в списке
4. Кликните на устройство для подключения
5. Следуйте инструкциям на экране

## Решение проблем

### Приложение не запускается

- Убедитесь, что у вас установлен Visual C++ Redistributable
- Проверьте, что антивирус не блокирует приложение
- Попробуйте запустить от имени администратора

### Устройство не находится

- Убедитесь, что устройство включено
- Проверьте, что Bluetooth включен на компьютере
- Попробуйте нажать кнопку "Обновить" в приложении

### Ошибки при работе

- Закройте и перезапустите приложение
- Перезагрузите устройство
- Проверьте подключение Bluetooth
