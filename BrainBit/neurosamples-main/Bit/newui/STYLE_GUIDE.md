# Style Guide for Modern UI Design

## Color Palette

- Primary Background: #FFFFFF (White)
- Secondary Background: #F5F7FA (Light Gray)
- Accent Colors:
  - Primary Blue: #007BFF (Blue)
  - Light Blue: #4DA1FF (Light Blue)
  - Teal: #20C997 (Teal/Turquoise)
- Text Colors:
  - Primary Text: #212529 (Dark Gray)
  - Secondary Text: #6C757D (Medium Gray)
  - Hint Text: #ADB5BD (Light Gray)
- Border Colors:
  - Default: #DEE2E6 (Light Border)
  - Focus: #007BFF (Blue)

## Typography

- Font Family: 'Segoe UI', 'Roboto', 'Helvetica Neue', sans-serif
- Font Sizes:
  - Headings: 24px, 20px, 18px
  - Body Text: 16px
  - Captions: 14px
  - Buttons: 16px

## Spacing & Layout

- Base Unit: 8px
- Padding/Margin Scale: 8px, 16px, 24px, 32px
- Border Radius: 8px (standard), 12px (cards), 24px (buttons)
- Shadow Levels:
  - Subtle: 0 1px 3px rgba(0,0,0,0.12), 0 1px 2px rgba(0,0,0,0.24)
  - Elevated: 0 3px 6px rgba(0,0,0,0.16), 0 3px 6px rgba(0,0,0,0.23)
  - Floating: 0 10px 20px rgba(0,0,0,0.19), 0 6px 6px rgba(0,0,0,0.23)

## Components

### Buttons

- Height: 48px
- Padding: 24px horizontal
- Border Radius: 24px
- Font Weight: 500
- States:
  - Default: Solid primary color with white text
  - Hover: Slightly darker shade + subtle shadow
  - Active: Pressed effect
  - Disabled: 50% opacity

### Cards

- Border Radius: 12px
- Padding: 24px
- Background: White
- Shadow: Subtle by default, elevated on hover

### Inputs

- Height: 48px
- Padding: 16px
- Border Radius: 8px
- Border: 1px solid #DEE2E6
- Focus State: 2px solid #007BFF

### Animations

- Transitions: 0.3s ease-in-out
- Loading Spinners: 1s linear infinite
- Page Transitions: Fade in/out 0.3s
