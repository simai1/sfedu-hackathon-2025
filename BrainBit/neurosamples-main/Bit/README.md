## Логирование данных с датчиков

В проект добавлена функция логирования всех данных с датчиков в консоль в формате JSON.
Данные включают:

- Сопротивление (Resistance)
- Сигналы (Signal)
- Данные об эмоциях (Emotion)
- Спектральные данные (Spectrum)

### Запуск логирования

Логирование автоматически включается при запуске соответствующих экранов в приложении:
- Экран сопротивления (Resistance Screen)
- Экран сигналов (Signal Screen)
- Экран эмоций (Emotion Screens)
- Экран спектра (Spectrum Screen)

### Автоматический запуск монополярных эмоций

После подключения устройства автоматически запускается режим монополярных эмоций с логированием всех данных.
Калибровка начинается автоматически без необходимости нажимать кнопку "Start".

### Формат данных

Все данные выводятся в консоль в формате JSON, готовом для отправки на сервер:

```json
{
  "timestamp": "2023-05-15T14:30:00.123456",
  "type": "signal",
  "data": {
    "O1": [15.2, 16.1, 14.8, 15.5, 16.0],
    "O2": [14.8, 15.5, 16.2, 15.0, 14.7],
    "T3": [16.0, 15.8, 16.5, 15.2, 15.9],
    "T4": [15.5, 16.2, 15.8, 16.1, 15.4]
  }
}
```

### Тестирование логирования

Для тестирования логирования можно использовать скрипт:

```bash
python test_sensor_logging.py
```

Для автоматического запуска монополярных эмоций после подключения устройства:

