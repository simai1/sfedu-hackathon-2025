ШАГ 1: СОЗДАНИЕ ДИСТРИБУТИВА
─────────────────────────────

Windows:
  1. Запустите: build.bat
  2. Запустите: create_distribution.bat
  3. Результат: папка distribution/BitApp/ и архив BitApp.zip

macOS:
  1. Запустите: ./build_macos.sh
  2. Запустите: ./create_distribution.sh
  3. Результат: архив BitApp-macOS.zip

ШАГ 2: ЗАГРУЗКА НА ВЕБ-САЙТ
────────────────────────────

1. Загрузите ZIP архив на ваш веб-сервер
2. Создайте страницу загрузки (см. download_page_example.html)
3. Разместите ссылки на скачивание

ШАГ 3: ЧТО ПОЛУЧИТ ПОЛЬЗОВАТЕЛЬ
────────────────────────────────

Пользователь скачает ZIP архив, который содержит:
  - BitApp.exe (Windows) или BitApp (macOS)
  - README.txt - подробная инструкция
  - ИНСТРУКЦИЯ.txt - краткая инструкция

Пользователь:
  1. Распакует архив
  2. Запустит BitApp.exe
  3. При первом запуске Windows/macOS может показать предупреждение
  4. Пользователь разрешит запуск
  5. Приложение готово к работе!

ВАЖНЫЕ МОМЕНТЫ:
───────────────

1. БЕЗОПАСНОСТЬ:
   - Windows SmartScreen будет показывать предупреждение
   - Это нормально для неподписанных приложений
   - Пользователь должен нажать "Выполнить в любом случае"

2. АНТИВИРУСЫ:
   - Некоторые антивирусы могут блокировать приложение
   - Рекомендуйте пользователям добавить папку в исключения

3. ПОДПИСЬ КОДА (опционально):
   - Можно подписать приложение сертификатом
   - Это уберет предупреждения Windows/macOS
   - Требует покупки сертификата

4. ОБНОВЛЕНИЯ:
   - Используйте версии в названии: BitApp-v1.0.zip
   - Обновляйте версию при каждом релизе
